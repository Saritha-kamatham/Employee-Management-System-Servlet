# Employee Management System

## Overview

The Employee Management System is a Java web application that allows users to manage employee records efficiently. It supports complete CRUD (Create, Read, Update, Delete) operations using Java Servlets, JDBC, and MySQL.

## Features

- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Responsive User Interface
- MySQL Database Integration
- JDBC Connectivity

## Technologies Used

- Java
- Jakarta Servlet
- JDBC
- MySQL
- HTML5
- CSS3
- Apache Tomcat
- Eclipse IDE
